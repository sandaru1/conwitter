Conwitter Options

  • Display
  • Tweet colors are combined to make a nicer effect. Choose your colors carefully! For example, a new tweet with your username mentioned will be displayed using a combination of both "New Tweet" and "Mention" colors.
  • Streaming API (Experimental)
  • Twitter offers an Alpha Streaming API, which allows push tweets. Tweets from your followers will appear as soon as those are published. You only need to enter username/password if you are using the streaming API. Everything else uses OAuth authentication system.

    Warning : Enabling this API might show you out of place tweets(ex : replies sent to users you follow by other users you don't follow). This is the default behaviour of the streaming API - an option will be given to filter this very soon.

  • Time Intervals(coming soon)
  • Make sure your time limits are reasonable. Twitter API only allows 150 API hits per hour.
